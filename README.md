#PUBLISHER

A nodejs application publisher (with docker)
Publish on a server with docker and node 6 image


##INSTALLATION

After cloning

```
& npm install -g
```

## USAGE

```
& cd my-nodejs-app
& publish --repository my-nodejs-app --user root --ip 10.0.0.0 --port 8888
```
